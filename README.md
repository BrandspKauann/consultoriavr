# Consultoria VR — Site

Site institucional da **Consultoria VR** (benefícios corporativos: vale refeição, Flash, Caju e conteúdo). Stack: **Vite**, **React**, **TypeScript**, **Tailwind CSS** e **shadcn/ui**.

**Produção:** [consultoriavr.com.br](https://www.consultoriavr.com.br)

---

## Repositório

```sh
git clone https://github.com/BrandspKauann/consultoriavr.git
cd consultoriavr
```

---

## Desenvolvimento local

1. Instale dependências: `npm install`
2. Configure variáveis de ambiente (veja `CONFIGURAR_ENV.md`). **Não commite segredos no `.env`.**
3. Suba o servidor: `npm run dev`

O Vite está configurado para **http://localhost:5174** (porta separada para não conflitar com outros projetos na 3000). O navegador pode abrir automaticamente.
